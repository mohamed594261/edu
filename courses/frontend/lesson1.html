1. ما هي لغة HTML؟

هي لغة ترميز النص التشعبي (HyperText Markup Language). ليست لغة برمجة بالمعنى التقليدي (لا تحتوي على منطق رياضي)، بل هي لغة وصفية تُخبر المتصفح كيف يوزع العناصر (نصوص، صور، جداول) داخل الصفحة.

2. الهيكل الأساسي للملف

كل صفحة ويب يجب أن تحتوي على هذه العناصر الثابتة:

    <!DOCTYPE html>  <!-- تعريف إصدار اللغة -->
    <html>           <!-- بداية الصفحة -->
        <head>       <!-- معلومات الصفحة (العنوان والترميز) -->
            <title>عنوان الصفحة</title>
        </head>
        <body>       <!-- المحتوى الظاهر للمستخدم -->
            محتوى الصفحة هنا...
        </body>
    </html>

3. أهم الوسوم (Tags) واستخداماتها

الوسم الوظيفة مثال
<h1> to <h6> عناوين بأحجام مختلفة

عنوان كبير
<p> كتابة فقرة نصية

هذا نص تجريبي للفقرة.
<a> إضافة رابط تشعبي رابط لجوجل
<img> إدراج صورة وسم يطلب مصدر الصورة (src)

4. أنواع القوائم

قائمة غير مرتبة (Unordered)

  • HTML
  • CSS
  • JavaScript

قائمة مرتبة (Ordered)

  1. تحميل محرر نصوص
  2. كتابة الكود
  3. فتح الملف بالمتصفح
